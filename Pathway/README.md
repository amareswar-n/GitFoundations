# GitFoundations – 30 Day Professional Git Roadmap
Project Scenario: SuperNova

This course simulates joining a real-world engineering team working on SuperNova.

You will progress from Junior Developer (Day 0) to Git Architect (Day 30).

Structure:
Week 1 – Foundations
Week 2 – Collaboration & Release Flow
Week 3 – Advanced Engineering
Week 4 – Expert & Recovery Engineering

Each phase includes:
- Concepts
- Commands
- Visual diagrams
- Lab exercises
- SuperNova Phase progression
